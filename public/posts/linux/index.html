Use Arch or Debian#

The two base distros in the Linux desktop realm

Download:

latest debian iso

latest arch iso

1: Free and open source#

You want to be able to fully trust you operating system? You can view every line of code of the operating system you’re running. Why would you pay for proprietary closed source bloat if you can easily install the Linux distro of your choice?

2: Community driven#

There is no company behind Linux The Linux foundation is not trying to make money but instead deliver a good, lightweight kernel

3: Package management#

Package management in windows is a mess. Linux package management is just clean

4: Lightweight#

Windows destroys your hardware over time by using ~4GB RAM in idle and using your disk as extended memory. Linux doesnt.

5: “I use Arch btw”#

You can tell everyone that you use Linux
